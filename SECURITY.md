# Security Policy

## Supported Versions

Security fixes are applied to the latest tagged BFF release.

## Reporting A Vulnerability

Please do not open a public issue for a suspected vulnerability. Report it
privately through
[GitHub private vulnerability reporting](https://github.com/vojtechkostal/BayesicForceFields/security/advisories/new).
If private vulnerability reporting is not enabled, email
vojtakostal8@gmail.com. Include the affected version, reproduction steps, and
expected impact.
