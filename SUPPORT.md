# Support

For installation questions, unexpected behavior, and feature requests, open a
[GitHub issue](https://github.com/vojtechkostal/BayesicForceFields/issues).

Include the BFF version, Python version, operating system, relevant
configuration fragment, and the complete error message. For GROMACS, CP2K, or
PLUMED issues, also include the external tool version and invocation.

Report security-sensitive issues privately as described in
[SECURITY.md](SECURITY.md).
